Curso Linux

COMO CRIAR UM USUÁRIO

criando usuario

$sudo adduser

Adicionando em um grupo

$sudo usermod -a -G sudo

logando no usuario

$su

COMO MONTAR UM PENDRIVE

Verifique o diretóro aonde o pendive foi identificado

user@linux:~$ fdisk -l

EX: dev/sdc1

Crie uma pasta para que o dispositivo seja montado

user@linux:~$ mkdir /media/usb-drive

Monte o pendrive indicando o diretório que ele está e o diretório reservado para montagem

user@linux:~$ mount dev/sdc1 /media/usb-drive/

Verifique o resultado

user@linux:~$ mount | grep sdc1

COMANDOS DIVERSOS

    • $uname -a (Mostra a distribuição e a versão do Kernel)
      $sudo adduser (cria um usuario)
      $sudo usermod -a -G [nome grupo] [nome usuario] (adiciona usuario em determinado grupo)
      $sudo su [nome usuario] //loga no perfil
      $sudo groupadd [opções] [nome do grupo]
      $echo (mostra o conteudo da variável de ambiente)
      $sudo who (mostra os logins de usuários)
      $sudo whoami (mostra os usuários logados)
      $ls /[diretorio] |grep [string] (filtra palafra chave no diretorio)
      $ls useradd -e 1999-12-31 (e = expire date, --expiredate; sintaxe ano/mês/dia)
      #lspci (mostra informaçoes do fabricante e modelo dos dispositivos PCIs conectados)
      $lsmod (lista os módulos carregados na memória)
      $modinfo [Nome_Modulo] (fornece as informações de um dado módulo)
      $insmod [Nome_arquivo] (carrega um módulo para a memória não resolve as dependências, ou seja outros softwares que podem ser necessários)
      #modprobe [Nome_modulo] (utilizado para carregar um módulo, carrega todas dependencias necessárias.*Recomendável>)
      #rmmod [Nome_modulo] (descarrega um módulo carregado na memória )
      $lsusb (lista os dispositivos USBs conectados)
      user@ip:/proc$ cat interrupts (Mosta canais que permite os que os outros programas façam requisições de interrupção ao processador)
      user@ip:/proc$ cat ioports (Mostra as portas de entrada e saída disponíveis para dispositivos)
      user@ip:/proc$ cat maininfo (Exibe informações da memória do dispositivo)
      $systemctl status [Serviço] (Exibe o status de determinado programa ou serviço instalado. Se não for passadop um serviço como argumento o comando mostra uma arvore com serviços do computador)
      $systemctl stop [Serviço] (Interrompe o funcionamento de determinado serviço)
      $systemctl start [Serviço] (Inicia determinado serviço)
      $systemclt enable [Serviço] (Desiguina um serviço para a carga de inicialização do sistema)
